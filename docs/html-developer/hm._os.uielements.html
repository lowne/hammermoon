Type eventName

Extends <#string>
An event name.

Valid event names are: "applicationActivated", "applicationDeactivated", "applicationHidden", "applicationShown", "mainWindowChanged", "focusedWindowChanged", "focusedElementChanged", "windowCreated", "windowMoved", "windowResized", "windowMinimized", "windowUnminimized", "elementDestroyed", "titleChanged".
